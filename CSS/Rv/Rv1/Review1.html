Combinator มี4แบบ

1.Descendant Selector ('')

จะเลือก element ที่อยู่ด้านในทั้งหมด เช่นตัวอย่างนี้



2. Child Selector (>)

จะเลือก element ที่อยู่ ด้านใน แต่จะไม่เลือก อันที่อยู่ ด้านในอีกทีนึง



3. Adjacent Sibling Selector(+)

เลือก element ที่อยู่ถัดจากตัวที่เลือก



4. General Sibling Selector (~)

เลือก element ที่อยู่ถัดจากตัวที่เลือก แต่จะไม่เหมือนกับอันที่แล้ว อันนี้จะเลือกทั้งหมด
