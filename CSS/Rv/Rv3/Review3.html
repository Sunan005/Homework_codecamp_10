2.grid และ flex แตกต่างกันอย่างไร

 Flex box ถูกออกแบบใหจัดการ layout แบบทิศทางเดียว คือ 1 มิติ
เช่น เรียงลำดับจากบนลงล่าง ซ้ายไปขวา เป็นต้น
แต่ Grid layout ถูกออกแบบมาเพื่อจัดการ layout แบบ 2 มิติ
คือ มีทั้งแบบแนวนอนและแนวตั้งในเวลาเดียวกัน หรือมองง่ายๆก็เป็นตาราง
